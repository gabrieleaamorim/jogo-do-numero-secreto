O aplicativo Jogo Secreto foi criado como objeto de treino para praticar a Lógica de programação em Typescript, usando o Node.js do lado do servidor.
Além das funcionalidades padrões, eu adicionei novos desafios, como:
- Input para colocar o nome do jogador;
- Transcrição do aplicativo por voz para deficientes visuais;
- Visibilidade das tentativas restantes para o jogador.
